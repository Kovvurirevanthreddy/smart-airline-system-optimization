✈️ Smart Airline Route Optimization

📌 Description
This project finds the most efficient flight route between airports using graph algorithms like Dijkstra and A*.

🚀 Features
- Shortest path calculation using Dijkstra Algorithm
- Intelligent route optimization using A* Algorithm
- Weighted graph representation of airports
- User input for source and destination
- Displays optimal route and total distance

🛠️ Tech Stack
- Python
- Heapq (Priority Queue)
- Graph Theory

📥 How to Run
1. open the notebook in google colab or jupyter notebook
2. Execute all cells
